# GAME RULES
🔗 <a href="https://bullet-hell-game.vercel.app/">_Bullet Hell Game 바로가기_</a>

## 1. 게임 개요

* 제목: **알파벳 피하기**
* 장르: **탄막 회피 생존 게임**
* 해상도: **1000 x 600**
* 플레이어 캐릭터: 잠수함 이미지 기반
* 목표: 사방에서 날아오는 프로그래밍 언어 탄막과 특수 오브젝트를 활용하며 최대한 오래 생존하고 높은 점수를 획득한다.

---

## 2. 기본 규칙

### 2.1 생명

* 시작 라이프: **5**
* 최대 라이프: **10**
* 라이프가 **0 이하**가 되면 게임 종료
* 게임 종료 시 랜덤한 **대학원 엔딩 멘트**가 표시된다.

### 2.2 점수

* 기본 상태: **초당 10점 획득**
* 슬로우 모드 중: **초당 5점 획득**
* 점수는 최종적으로 **소수점 이하 버림** 처리
* 게임 종료 후 플레이어 이름을 입력해 **랭킹 TOP 10**에 등록 가능

### 2.3 랭킹

* 저장 방식: **localStorage**
* 최대 저장 수: **10개**
* 이름 입력 제한: **16자**

---

## 3. 조작 방식

### 3.1 이동

* `← ↑ ↓ →` 방향키로 이동
* 플레이어는 맵 바깥으로 나갈 수 없다.

### 3.2 대시

* `Shift + 방향키`
* 현재 이동 방향으로 **짧은 거리 대시**
* 시작 대시 횟수: **3회**
* 스킬 탄막 획득 시 대시 횟수 **+1**
* 대시 사용 시 하늘색 잔상 효과가 잠깐 출력된다.

### 3.3 슬로우 모드

* 키: `Space`
* 시작 시 슬로우 사용 가능 시간: **5초**
* 슬로우 사용 중:

  * 시간 흐름 속도 감소
  * 플레이어 이동 속도 감소
  * 장애물 속도 감소
  * 점수 획득량: **초당 5점**
* 사용 가능 시간이 0초가 되면 자동 해제
* 슬로우가 꺼져 있는 동안 **1초당 1초씩 충전**
* 슬로우 사용 가능 시간은 플레이어 주변의 **보라색 링**으로 항상 표시된다.

---

## 4. 일반 탄막 규칙

### 4.1 기본 장애물

* 맵 테두리에서 랜덤 생성
* 프로그래밍 언어 이름 기반의 **이미지 탄막**으로 등장
* 플레이어와 충돌 시 라이프 **1 감소**

### 4.2 피격 시 효과

* 플레이어는 피격 후 **1.5초 동안 무적**
* 같은 시간 동안 **블링크 효과** 적용
* 플레이어 주변에 **노란 타이머 링**이 표시되며 남은 시간에 따라 감소

---

## 5. 특수 탄막 / 아이템

### 5.1 Heart Bullet

* 효과: 라이프 **+1**
* 플레이어와 충돌 시 해당 위치에 **`❤️+1`** 텍스트가 약 **0.3초간 표시**

### 5.2 Shield Bullet

* 효과: 플레이어에게 **실드 적용**
* 기본 지속시간: **3초**
* 실드 상태에서는 일반 탄막 충돌 시 라이프 감소 없음
* 실드 상태는 다음 시각 효과를 가진다:

  * 플레이어 주변 노란 glow
  * 플레이어 주변 노란 타이머 링
  * 화면 테두리 노란 border 타이머
* 실드 시간은 **남은 시간 기반**으로 관리된다.
* 슬로우 모드 중 실드가 켜져 있으면:

  * 남은 실드 시간 감소 속도가 **50%**로 줄어든다.
* 슬로우 모드 중 실드 탄막을 먹어도:

  * 여전히 **3초**가 추가되며
  * 그 이후 감소 속도는 슬로우 규칙에 따라 **절반 속도**로 적용된다.

### 5.3 Rainbow Bullet

* 효과:

  * 화면 중앙에 **RAINBOW** 텍스트 표시
  * 현재 화면의 **일반 탄막 전부 제거**
* 하트, 실드, 스킬, 감속 탄막 같은 특수 오브젝트는 유지 가능

### 5.4 Skill Bullet

* 색상: **sky blue**
* 효과: 대시 횟수 **+1**

### 5.5 Freeze Bullet

* 색상: **sky blue / ice 계열**
* 효과: 장애물 속도 감소
* 적용 방식:

  * 획득 시 장애물 속도 배율 감소
  * 속도는 항상 **0보다 큰 값 유지**
  * 최소 배율은 **35%** 이하로 내려가지 않음

---

## 6. 히든 보너스 패턴

### 6.1 발동 조건

* 플레이어 라이프가 **10**에 도달하면 발동

### 6.2 패턴 시작 시

* 라이프 **5 감소**
* 보너스 패턴 지속시간: **5초**
* 화면 테두리가 **노란색**으로 변경
* border는 남은 시간에 따라 **시계 방향으로 점점 사라짐**

### 6.3 패턴 진행 중

* 모든 탄막과 플레이어가 충돌할 때마다 **30점 추가 획득**
* 충돌해도 라이프 감소 없음

### 6.4 패턴 종료 시

* 플레이어에게 **실드 효과 3초 자동 부여**

---

## 7. 시각 효과 정리

### 7.1 플레이어 주변 링

* **보라색 링**: 슬로우 사용 가능 시간
* **노란 링 1**: 실드 지속 시간
* **노란 링 2**: 피격 후 무적 시간

### 7.2 화면 테두리 효과

* **노란 border**: 실드 지속 시간
* **노란 bonus border**: 히든 보너스 패턴 시간
* 남은 시간에 따라 시계 방향으로 줄어드는 표현 사용

### 7.3 팝업 및 텍스트

* Heart 획득: `❤️+1`
* Rainbow 획득: `RAINBOW`
* 게임 종료: 랜덤 엔딩 멘트

---

## 8. UI 표시 요소

* LIFE: 하트 이모지로 표시

  * 예: `❤️❤️❤️❤️❤️`
* SCORE: 정수 점수
* TIME: 생존 시간
* SLOW: 현재 슬로우 모드 상태 및 남은 충전 시간
* DASH: 남은 대시 횟수
* 장애물 속도 %
* 실드 남은 시간
* 피격 무적 남은 시간
* 보너스 패턴 남은 시간

---

## 9. 시스템 안정성 규칙

### 9.1 입력 초기화

* `Alt + Tab` 또는 창 포커스 이탈 시 입력 상태 초기화
* 초기화 대상:

  * 방향키
  * Shift
  * Space
  * 슬로우 모드 상태
  * Shift 연속 입력 판정 상태

### 9.2 다시하기 버튼

* 게임 진행 중에는 **비활성화**
* 게임 종료 후에만 다시 시작 가능

---

## 10. 요약

이 게임은 단순 회피 게임이 아니라, 다음 리소스를 조합하는 생존 게임이다.

* 생명 관리
* 슬로우 자원 관리
* 실드 지속시간 관리
* 대시 횟수 관리
* 장애물 속도 제어
* 히든 보너스 패턴 진입 전략

즉, 오래 버티는 것뿐 아니라 **언제 하트, 실드, 슬로우, 대시, 감속, 무지개를 쓰느냐**가 핵심 전략 요소다.
